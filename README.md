# 🎭 装个文化人 (AGE-AI)

> **"然今乱局，尔其能独善其身乎？"** —— 你的 AI 雅言翻译官。

## 🌟 项目亮点
- **四海文风**：庙堂之高、江湖之远、闺阁之情、市井之趣，随心切换。
- **炼字秘法**：基于国产大模型精调，拒绝生搬硬套，追求信达雅。
- **万民广场**：看全网各路豪杰如何用古文吐槽生活。
- **密函保护**：不想公开的真心话，唯有数据库与你知晓。

## 🛠️ 技术栈
- **Frontend**: Next.js 14 + Shadcn UI + Tailwind CSS
- **Backend**: Supabase (Database & Auth)
- **AI Core**: DeepSeek / Qwen API
- **Deployment**: Vercel
